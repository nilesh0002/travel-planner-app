# TravelPlanner 🌍

A simple and beautiful travel planning application built with React, Node.js, and Supabase.

## Features
- **Trip Management**: Create and track your upcoming adventures.
- **Itinerary Organizer**: Add activities, flights, and hotels to your daily plan.
- **Beautiful UI**: Modern glassmorphism design with responsive layout.
- **Authentication**: Secure login and session management via Supabase.

## Tech Stack
- **Frontend**: React (TypeScript), CSS3 (Flexbox/Grid), Vite.
- **Backend**: Node.js, Express.
- **Database**: Supabase (PostgreSQL).

## Project Structure
- `/client`: React frontend with TypeScript.
- `/server`: Node.js + Express backend.
- `database.sql`: SQL schema for the database tables.

## How to Run Locally

### 1. Backend Setup
1. Navigate to the `server` directory.
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example`.
4. Start the server: `npm run dev`

### 2. Frontend Setup
1. Navigate to the `client` directory.
2. Install dependencies: `npm install`
3. Create a `.env` file based on the required Supabase URL and Key.
4. Start the app: `npm run dev`

## Concepts Used
This project naturally applies fundamental web development concepts:
- **HTML/CSS**: Semantic structure, Flexbox, and Media Queries for responsiveness.
- **JavaScript/TypeScript**: Async/await for APIs, interface definitions, and state management.
- **DOM**: Dynamic rendering and event handling.
- **Node.js**: Modular architecture with routes, controllers, and services.
- **API**: RESTful endpoints with proper status codes.
